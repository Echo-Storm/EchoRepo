EchoRepo

Repository for Echo-Storm's custom Kodi addons, skins, and tools

Installation

  1. Download the repository zip: repository.echostorm-1.0.0.zip
  2. In Kodi, go to Settings → Add-ons → Install from zip file
  3. Browse to and select the downloaded zip file
  4. Wait for the "Add-on installed" notification

Alternative: Install via URL

Repository URL: https://raw.githubusercontent.com/Echo-Storm/EchoRepo/main

Included Add-ons

Check the addons.xml file for the current list of available add-ons.
